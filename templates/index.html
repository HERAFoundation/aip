Welcome to AIP

ATProtocol Identity Provider - Manage your OAuth and app-passwords

About AIP

AIP is a web application that manages ATProtocol access tokens using OAuth and app-passwords. It provides a secure and convenient way to authenticate and authorize applications that interact with the ATProtocol ecosystem.

Features

  • OAuth authentication flow support
  • App-password management
  • ATProtocol identity resolution
  • Secure token management

Getting Started

To get started with AIP, navigate to the Identity page to manage your ATProtocol identities and authentication methods.
